CyberMap 3D

Accueil Visualisation Documentation

Documentation

CyberMap 3D est compose de plusieurs outils Python et d'une interface web 3D. Chaque outil joue un role dans la chaine de securite — du scan reseau jusqu'a la visualisation. Clique sur un outil pour voir les details.

Scan & Reconnaissance
scanner.py ▶
Scan reseau simple par ping
Python

Utilise nmap en mode ping (-sn) pour detecter toutes les machines connectees sur le reseau local. Rapide et discret, il ne scanne pas les ports.

scanner.scan(hosts="192.168.0.0/24", arguments="-sn")

Resultat : liste des IPs actives sauvegardee dans data/reseau.json

scanner_ports.py ▶
Scan avance — ports, services, OS
Python

Scan approfondi avec detection des ports ouverts (-sV), identification de l'OS (-O) et des versions des services. Plus lent mais beaucoup plus informatif.

scanner.scan(hosts=ip, arguments="-sV -O --top-ports 20")

Detecte par exemple : Windows 11, ports 135/139/445 ouverts sur ton PC.

osint.py ▶
Collecte d'informations publiques
Python

OSINT (Open Source Intelligence) — collecte des informations publiques sur une cible sans l'attaquer. Geolocalisaton IP, headers HTTP, technologies utilisees.

requests.get("http://ip-api.com/json/" + ip)

Exemple : google.com → IP 172.217.22.46 → Frankfurt, Allemagne, Google LLC

Detection & Alertes
vulnerabilites.py ▶
Analyse des failles CVE
Cybersecurite

Interroge la base de donnees CVE du NIST (gouvernement americain) pour trouver les failles connues sur les services detectes. Chaque faille a un score de 0 a 10.

requests.get("https://services.nvd.nist.gov/rest/json/cves/2.0?keywordSearch=" + service)

Score CRITICAL (9-10) = faille tres dangereuse. Le port 445 ouvert sur ton PC correspond a la faille WannaCry (CVE-2017-0144, score 9.3).

detecteur.py ▶
Systeme de detection d'intrusion (IDS)
Cybersecurite

Compare le reseau actuel avec une liste blanche de machines connues. Si une nouvelle machine apparait, une alerte est generee et sauvegardee dans alertes.json.

if ip not in machines_connues: alertes.append({"type": "INTRUS", "ip": ip})

C'est le principe d'un vrai IDS comme Snort ou Suricata utilise en entreprise.

Simulations d'attaques
honeypot.py ▶
Faux serveur piege
Cybersecurite

Simule 3 faux serveurs (SSH port 2222, HTTP port 8080, FTP port 2121) pour attirer et enregistrer les tentatives d'intrusion. Utilise le threading pour ecouter les 3 ports simultanement.

t1 = threading.Thread(target=honeypot_ssh, daemon=True) t2 = threading.Thread(target=honeypot_http, daemon=True) t3 = threading.Thread(target=honeypot_ftp, daemon=True)
craqueur.py ▶
Craquage de hash MD5/SHA
Cybersecurite

Attaque par dictionnaire — teste une liste de mots de passe courants et compare leur hash avec la cible. Demontre pourquoi "admin" se craque en moins d'1 seconde.

hash_essai = hashlib.md5(mot.encode()).hexdigest() if hash_essai == hash_cible: print("Trouve :", mot)
chiffrement.py ▶
Chiffrement AES-256
Cybersecurite

Implementation du chiffrement AES-256 en mode CBC. Chiffre et dechiffre des messages avec une cle secrete. Sans la cle, le message est impossible a lire.

cipher = AES.new(cle, AES.MODE_CBC, iv) message_chiffre = cipher.encrypt(pad(message, AES.block_size))

HTTPS, WhatsApp et Signal utilisent AES pour proteger toutes vos communications.

Visualisation 3D
index.html ▶
Interface 3D interactive
Three.js

Visualisation 3D du reseau avec Three.js. Charge les modeles SolidWorks (.glb), place les machines en cercle, anime les connexions et simule la propagation d'une attaque.

const loader = new THREE.GLTFLoader(); loader.load("pc.glb", (gltf) => scene.add(gltf.scene));

Fonctionnalites : rotation automatique, clic sur machine, pare-feu, radar, log d'evenements, particules de trafic.

Modeles SolidWorks ▶
PC, Serveur, Routeur en 3D
SolidWorks

3 modeles 3D crees dans SolidWorks et exportes en format .glb (compatible Three.js). Le format glTF est le standard web pour les modeles 3D — compact et rapide a charger.

Fichiers : pc.glb (62kb), routeur.glb (13kb), serveur.glb (9kb)

Generateur de rapport PDF
Python

Genere automatiquement un rapport PDF professionnel avec ReportLab. Resume les machines detectees, ports ouverts, vulnerabilites et recommandations de securite.

doc = SimpleDocTemplate("rapport.pdf", pagesize=A4) doc.build(contenu)
