Le principe général
Ce script fonctionne comme un gardien de sécurité qui connaît tous les habitants d'un immeuble. Chaque fois qu'il fait sa ronde, il vérifie si quelqu'un de nouveau est entré sans autorisation.

1er lancement → il scanne le réseau et enregistre toutes les machines comme "connues" dans un fichier.

Lancements suivants → il rescanne et compare avec la liste. Si une nouvelle machine apparaît = alerte intrus !
Les 3 scénarios possibles
Machine connue
→
✓ Normal, rien à signaler
Nouvelle machine
→
🚨 ALERTE — intrus potentiel !
Machine disparue
→
⚠ Appareil éteint ou déconnecté
Les deux fichiers de données
FICHIER_CONNUS = "machines_connues.json"
FICHIER_ALERTES = "alertes.json"
machines_connues.json → la liste blanche. Toutes les machines autorisées sur le réseau. Ce fichier est créé au 1er lancement et mis à jour à chaque scan.

alertes.json → le journal des intrusions. Chaque fois qu'une machine inconnue est détectée, elle est enregistrée ici avec l'heure exacte.

Le module os permet d'interagir avec le système d'exploitation — lire des fichiers, vérifier leur existence, naviguer dans les dossiers.

os.path.exists() retourne True si le fichier existe, False sinon. On l'utilise pour savoir si c'est le premier lancement ou non.
datetime — horodater les alertes
from datetime import datetime
datetime.now().strftime("%d/%m/%Y %H:%M:%S")
# → "31/05/2026 14:23:45"
datetime.now() donne la date et l'heure exacte. strftime la formate en texte lisible. Chaque alerte est ainsi horodatée — essentiel pour un vrai journal de sécurité.
Lien avec la vraie cybersécurité
Ce que tu construis s'appelle un IDS — Intrusion Detection System. C'est un des outils fondamentaux de la cybersécurité en entreprise.

Les vrais IDS comme Snort ou Suricata font exactement la même chose mais en temps réel et avec des règles bien plus complexes.

Ton détecteur fait :
→ Inventaire du réseau
→ Comparaison avec la liste blanche
→ Génération d'alertes horodatées
→ Sauvegarde du journal

C'est la base de tout système de surveillance réseau professionnel. En mentionnant "IDS" dans ton dossier Parcoursup, les jurys de cybersécurité verront que tu connais les concepts réels du domaine.
