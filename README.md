# 🔐 CyberMap 3D

Visualisateur interactif de cyberattaques réseau en 3D.

## 📌 Présentation

CyberMap 3D est un projet personnel réalisé en classe de Première MP3D.
Il simule la propagation d'une cyberattaque sur un réseau informatique
et la visualise en temps réel dans un environnement 3D interactif.

## 🛠 Technologies utilisées

- **Python** — simulation de la propagation du virus sur le réseau
- **JSON** — format d'échange de données entre Python et le web
- **Three.js** — visualisation 3D interactive dans le navigateur
- **SolidWorks** — modélisation 3D des machines du réseau (PC, serveur, routeur)
- **HTML / CSS** — interface utilisateur

## 🎯 Fonctionnement

1. Le script Python simule un réseau de 4 machines
2. Une cyberattaque est lancée sur une machine cible
3. Le virus se propage automatiquement aux autres machines
4. L'état du réseau est exporté en JSON
5. La page web charge les modèles 3D et visualise l'attaque en direct

## 🚀 Lancer le projet

1. Cloner le dépôt
2. Lancer le serveur local :
3. Ouvrir `http://ryderbailleux3-stack.giyhub.io/cybermap3d/accueil.html` dans le navigateur
4. Cliquer sur **"Lancer l'attaque"**
