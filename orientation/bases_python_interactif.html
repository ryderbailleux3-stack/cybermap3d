Cours interactif — les bases de Python

Concept Une variable, c'est quoi ?
Une variable est une boîte dans la mémoire de l'ordinateur. Tu lui donnes un nom, tu y mets une valeur. Tu peux la lire ou la modifier à tout moment.
prenom = "Hugo"
age = 17
note = 15.5
print(prenom)
résultat :
Hugo
Le signe = ne veut pas dire "égal" en Python. Il veut dire "met la valeur de droite dans la boîte de gauche".
Règles Nommer une variable
Les noms de variables suivent 3 règles simples.
# Valide
mon_age = 17
nomUtilisateur = "Hugo"
score2 = 100
# Invalide
2score = 100  # commence par un chiffre
mon age = 17  # espace interdit
Convention en Python : on utilise les_underscores pour séparer les mots. Ex : nom_machine, vitesse_max.
Concept Les types de données
Chaque variable a un type — la nature de ce qu'elle contient. Python en gère 4 principaux.
# int — nombre entier
age = 17
# float — nombre décimal
moyenne = 14.75
# str — texte (string)
prenom = "Hugo"
# bool — vrai ou faux
infecte = False
connecte = True
Pour savoir le type d'une variable : print(type(age)) → affiche <class 'int'>
Opérations Faire des calculs
a = 10
b = 3
print(a + b)  # addition → 13
print(a - b)  # soustraction → 7
print(a * b)  # multiplication → 30
print(a / b)  # division → 3.33
print(a % b)  # reste → 1
print(a ** b) # puissance → 1000
Concept Une liste, c'est quoi ?
Une liste stocke plusieurs valeurs dans une seule variable. Les éléments sont dans l'ordre et chacun a un numéro (index) qui commence à 0.
fruits = ["pomme", "banane", "cerise"]
print(fruits[0]) # pomme
print(fruits[1]) # banane
print(fruits[2]) # cerise
print(len(fruits)) # 3 (taille)
L'index commence toujours à 0, pas à 1. Le premier élément = index 0, le deuxième = index 1, etc.
Opérations Modifier une liste
machines = ["PC", "Serveur"]
machines.append("Routeur") # ajouter
machines.remove("PC") # supprimer
machines[0] = "Laptop" # modifier
print(machines)
résultat :
['Laptop', 'Routeur']
Concept if / elif / else
Une condition permet à Python de choisir quoi faire selon une situation. C'est la base de toute logique en programmation.
note = 14
if note >= 16:
    print("Très bien")
elif note >= 10:
    print("Passable")
else:
    print("Insuffisant")
résultat :
Passable
elif = "sinon si". Tu peux en mettre autant que tu veux entre le if et le else.
Opérateurs Comparer des valeurs
a == b # égal à
a != b # différent de
a > b # supérieur à
a < b # inférieur à
a >= b # supérieur ou égal
a <= b # inférieur ou égal
and # les deux doivent être vrais
or # au moins un vrai
not # inverse vrai/faux
Concept La boucle for
La boucle for répète un bloc de code pour chaque élément d'une liste (ou d'une plage de nombres).
# Parcourir une liste
machines = ["PC", "Serveur", "Routeur"]
for m in machines:
    print(m)
# Compter de 0 à 4
for i in range(5):
    print(i)
résultat (liste) :
PC
Serveur
Routeur
Concept La boucle while
La boucle while répète tant qu'une condition est vraie. Utile quand on ne sait pas à l'avance combien de fois répéter.
compteur = 0
while compteur < 3:
    print(f"Tour numéro {compteur}")
    compteur += 1
résultat :
Tour numéro 0
Tour numéro 1
Tour numéro 2
compteur += 1 est un raccourci pour compteur = compteur + 1. Sans ça, la boucle tourne à l'infini !
Concept Créer une fonction
Une fonction est un bloc de code réutilisable qu'on définit une fois et qu'on appelle autant de fois qu'on veut. Elle peut recevoir des paramètres et retourner un résultat.
def saluer(prenom):
    print(f"Bonjour {prenom} !")
saluer("Hugo")
saluer("Alice")
résultat :
Bonjour Hugo !
Bonjour Alice !
return Retourner une valeur
return permet à la fonction de renvoyer un résultat qu'on peut stocker dans une variable.
def additionner(a, b):
    return a + b
resultat = additionner(5, 3)
print(resultat)
résultat :
8
Concept Un dictionnaire, c'est quoi ?
Un dictionnaire stocke des données sous forme de clé : valeur. Comme une vraie fiche d'identité — chaque info a un label.
machine = {
    "nom": "PC-Bureau",
    "ip": "192.168.1.10",
    "infecte": False
}
print(machine["nom"])
print(machine["ip"])
résultat :
PC-Bureau
192.168.1.10
Opérations Modifier un dictionnaire
# Modifier une valeur
machine["infecte"] = True
# Ajouter une nouvelle clé
machine["type"] = "serveur"
# Supprimer une clé
del machine["infecte"]
# Vérifier si une clé existe
if "ip" in machine:
    print("IP trouvée !")
Dans ton projet, chaque machine du réseau est un dictionnaire. C'est pour ça qu'on écrit machine["nom"] ou machine["infecte"].
